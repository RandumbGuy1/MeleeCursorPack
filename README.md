# The Melee Cursor Pack
Basic cursor pack for Windows 10/11 themed around Super Smash Bros. Melee

## How to Use (Windows 11)
- Download the Melee Cursor pack .zip
- Go to Settings > Accessibility > Mouse Pointer and touch > Customize Pointer Image
- Remap all cursors to the .cur file corresponding to their name

## Pack overview
- **Style:** Pixelated
- **Target size:** 32×32 (standard)
- **File types:** .cur (static) and .png
- **Compatibility:** Windows 10 and Windows 11 pointer schemes.
- **Precision:** Hotspots matched to Windows defaults to avoid misclicks.

## Hotspot alignment reference
- **Normal/help select:** Tip of the pointer fingers.
- **Link select:** Tip of the select circle.
- **Text select (I‑beam):** Left edge of the super scope.
- **Precision select:** Exact center of crosshair.
- **Unavailable:** Exact center of the circle.
- **Move/resize:** Geometric center of the icon.
